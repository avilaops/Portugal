# Processo de Visto para Portugal

## Status Atual
- **Titular**: Nícolas
- **Vistos Existentes**: Estados Unidos, África do Sul, Canadá
- **Residência Atual**: Brasil

## Opções de Visto para Empreendedor

### 1. Visto D2 - Empreendedor/Investidor
- **Requisito**: Plano de negócios viável
- **Investimento mínimo**: Não especificado formalmente, mas recomenda-se demonstrar capital suficiente
- **Duração inicial**: 4 meses (para entrada), renovável para 2 anos
- **Vantagens**: Permite residir e trabalhar em Portugal

### 2. Visto D7 - Rendimentos Próprios
- **Requisito**: Renda passiva ou rendimentos comprovados
- **Duração inicial**: 4 meses (para entrada), renovável para 2 anos

## Documentação Necessária (Checklist)

### Documentos Pessoais
- [ ] Passaporte válido (mínimo 3 meses após data prevista de saída)
- [ ] Certidão de Nascimento apostilada
- [ ] Certidão de Antecedentes Criminais (Brasil e países onde residiu)
- [ ] Comprovante de meios de subsistência

### Documentos Empresariais (Visto D2)
- [ ] Plano de negócios detalhado
- [ ] Comprovação de capital/recursos financeiros
- [ ] Contrato de constituição da empresa (se já existir)
- [ ] Projeções financeiras
- [ ] Cartas de intenção de clientes/parceiros (opcional mas recomendado)

### Outros Documentos
- [ ] Seguro de saúde válido em Portugal (PB4 ou seguro privado)
- [ ] Comprovante de alojamento em Portugal
- [ ] Formulário de pedido de visto preenchido
- [ ] Fotos 3x4 recentes
- [ ] Pagamento das taxas consulares

## Timeline Estimado
1. **Preparação de documentos**: 1-2 meses
2. **Agendamento no consulado**: 2-4 semanas
3. **Análise do pedido**: 2-3 meses
4. **Entrada em Portugal**: Até 4 meses após aprovação
