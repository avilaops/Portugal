## How to Be Added

Once you have a merged pull request:
1. You'll automatically appear in GitHub's contributor list
2. Add yourself to this file in your PR
3. Include: Name, GitHub username, brief contribution description

Format:
```markdown
### Your Name (@github-username)
- **Contributions**:
  - Brief description of your contributions
  - Can be multiple bullet points
```
